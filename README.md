acm-conventions
=========

Корень содержит файлы с базовыми соглашениями по кодированию. basic.cpp будет разбит на разные файлы, чтобы было легко просматривать по темам.
Каждый образец будет решением простой (понятной для всех) задачи.
До того, чтобы мы хотели одинаково назвать переменные, еще далеко, 
но к этому надо стремиться. У файлов должны быть говорящие названия, 
чтобы можно было легко к ним обращаться в случае неопределенности.

Все решения/алгоритмы оформляются на основе template.cpp
Он должен быть протестирован на MS C++ 2008 (желательно, каких я только ни видел), GNU C++ ~4.6.2 - 4.8.1 (или MinGW)

Принятые реализации алгоритмов лежат в Algorithms. Начинаться должны с названия категории (Graph_xxx, Geom_xxx, DP_xxx, ...)

Решения будут заливаться в папку Solutions по контестам или архивам (а может навалом, посмотрим как приживется). Начинаться должны с первой буквы фамилии кодировщика, или T, если писали вместе (чтобы после фразы "посоны смарите" искать было проще). Можно таким образом назвать какую-нибудь папку и туда скалдывать уже 
как хочешь
